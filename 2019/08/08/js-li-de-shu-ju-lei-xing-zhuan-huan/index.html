1 任意类型转字符串

1.1 使用String() 方法

String(1) //"1"
String(true) //"true"
String(null) //"null"
String(undefined) //"undefined"
String({}) //"[object Object]"

1.2 使用toString()方法

(1).toString()
"1"
true.toString()
"true"
null.toString()
// 无法转换 报错 : Uncaught TypeError: Cannot read property 'toString' of null
undefined.toString()
// 无法转换 报错 : Uncaught TypeError: Cannot read property 'toString' of undefined
({}).toString()
"[object Object]"

2 任意类型转换数字

2.1 使用Number()

Number(true) // 1
Number(null) // 0
Number(undefined) // NaN
Number({}) // NaN
Number("1") // 1

2.2 使用parseInt

用于将字符串转化为数字, 第一个参数为要转化的字符串，第二个参数为进制，例子中为十进制

parseInt("1", 10) // 1
parseInt("1a", 10) // 1
parseInt("a1", 10) // NaN

2.3 使用parseFloat

用于将字符串转化为浮点数

2.4 使用x - 0

'18' - 0
18
true - 0
1
false - 0
0

3 任意类型转布尔值

3.1 使用Boolean()

Boolean(1) // true
Boolean(NaN) // false
Boolean(0) // false
Boolean('') // false
Boolean('1') // true
Boolean(null) // false
Boolean(undefined) // false

3.2 使用!!x

!!'' // false
!!NaN // false
